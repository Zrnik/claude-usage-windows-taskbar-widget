Do CredentialStore.cs přidat (ve stejném namespace ClaudeUsageWidgetProvider):

2. Record `AccountInfo` (internal sealed) s parametry: `ServiceType Service`, `OAuthCredential Credential`, `string Label`.

3. Metodu `LoadAllAccounts()` (public static, vrátí `List<AccountInfo>`):
   - Volá existující `LoadAllCredentials()` pro Claude credentials
   - Deduplikuje přes `HashSet<string> seen` — klíč = `"claude:" + token[..Math.Min(32, token.Length)]`
   - Label: pokud `cred.SourcePath.StartsWith("wsl:")` → "claude-wsl", jinak "claude-windows"
   - Volá novou `LoadCodexCredentials()` pro Codex
   - Deduplikuje Codex stejně — klíč = `"codex:" + token[..Math.Min(32, token.Length)]`
   - Vrátí seznam: Claude účty první, pak Codex

4. Privátní metodu `LoadCodexCredentials()` (vrátí `List<AccountInfo>`):
   - Volá `TryLoadCodexWindowsCredential()` → přidá do result pokud != null
   - Volá `TryLoadCodexWslCredential()` → přidá do result pokud != null
   - Vrátí result

5. Privátní metodu `TryLoadCodexWindowsCredential()` (vrátí `AccountInfo?`):
   - Path: `Environment.GetFolderPath(SpecialFolder.UserProfile) + "\.codex\auth.json"`
   - Pokud soubor neexistuje: return null
   - `try { var json = File.ReadAllText(path); var cred = ParseCodexCredentialJson(json, path); return cred != null ? new AccountInfo(ServiceType.Codex, cred, "codex-windows") : null; } catch { return null; }`

6. Privátní metodu `TryLoadCodexWslCredential()` (vrátí `AccountInfo?`):
   - `try { var json = RunWsl("cat ~/.codex/auth.json"); if (string.IsNullOrWhiteSpace(json)) return null; var cred = ParseCodexCredentialJson(json, "wsl:~/.codex/auth.json"); return cred != null ? new AccountInfo(ServiceType.Codex, cred, "codex-wsl") : null; } catch { return null; }`

7. Privátní metodu `ParseCodexCredentialJson(string json, string sourcePath)` (vrátí `OAuthCredential?`):
   - `JsonNode.Parse(json)` — pokud null: return null
   - Token: zkusit `doc["access_token"]`, pak `doc["accessToken"]`, pak `doc["token"]` — GetValue&lt;string&gt;() — pokud všechny null nebo prázdné: return null
   - RefreshToken: zkusit `doc["refresh_token"]` ?? `doc["refreshToken"]` ?? `""` — prázdný string pokud chybí
   - ExpiresAt: zkusit `doc["expires_at"]` ?? `doc["expiresAt"]` — pokud existuje:
     - Nejprve zkusit `GetValue<long>()` (epoch ms)
     - Pokud selže (InvalidOperationException), zkusit `DateTimeOffset.TryParse(node.GetValue<string>(), out var dto)` → `dto.ToUnixTimeMilliseconds()`
     - Pokud obojí selže: `long.MaxValue`
     - Pokud node neexistuje: `long.MaxValue` (API key = nikdy nevyprší)
   - Celé v try/catch → při výjimce return null
   - Vrátit `new OAuthCredential { AccessToken = token, RefreshToken = refreshToken, ExpiresAt = expiresAt, SourcePath = sourcePath }`
