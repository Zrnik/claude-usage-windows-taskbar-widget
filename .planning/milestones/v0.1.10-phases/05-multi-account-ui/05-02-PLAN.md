# Plan 05-02: AccountPanel UserControl + multi-account layout + dynamická šířka

## Objective

Přepsat MainWindow tak, aby zobrazoval N AccountPanel sloupců horizontálně a dynamicky měnil šířku podle počtu účtů. Každý AccountPanel zobrazuje ikonu služby vlevo od dvou progress barů.

## Context

**Aktuální stav:**
- `MainWindow.xaml`: fixní `Width="170"`, 2 ProgressBary přímo v okně, žádná ikona
- `MainWindow.xaml.cs`: konstruktor přijímá `ClaudeApiClient apiClient` (1 účet)
- `Program.cs`: vytváří jedno `MainWindow(clients[0], ...)` — zobrazuje pouze první účet
- `MainWindow` obsahuje `PositionWindow()` která zná tray width a umí spočítat X pozici

**Záměr:**
- Nový `AccountPanel` UserControl = jeden sloupec (ikona + Bar5h + Bar7d)
- `MainWindow` dostane `List<ClaudeApiClient>` a dynamicky vytvoří N `AccountPanel` instancí
- Šířka okna = `N × 170` (nebo jiná konstanta per-column) — stávající tray-aware positioning zůstává
- `PositionWindow()` v MainWindow bude používat `Width` (nově dynamické) pro výpočet X — kód se nemusí měnit

**Konstanty:**
- `COL_WIDTH = 170` (stávající šířka pro 1 účet)
- `ICON_WIDTH = 22` (ikona 20px + 2px margin)
- Efektivní šířka sloupce s ikonou: bary se zúží o `ICON_WIDTH`, celková šířka sloupce zůstane 170

**AccountPanel.xaml.cs** — logika:
```csharp
using System.Windows.Controls;
using System.Windows.Media;
using System.Windows.Media.Imaging;

namespace ClaudeUsageWidgetProvider;

public partial class AccountPanel : UserControl
{
    private static readonly string[] SpinnerFrames = ["|", "/", "—", "\\"];
    private int _spinnerFrame;

    public AccountPanel(ServiceType service)
    {
        InitializeComponent();
        ServiceIcon.Source = new BitmapImage(new Uri(
            service == ServiceType.Claude
                ? "pack://application:,,,/Assets/claude-logo.png"
                : "pack://application:,,,/Assets/codex-logo.png"));
    }

    public void UpdateBars(UsageData data)
    {
        Text5h.Foreground = Brushes.White;
        Text7d.Foreground = Brushes.White;

        var first = data.Limits.ElementAtOrDefault(0);
        var second = data.Limits.ElementAtOrDefault(1);

        if (first != null)
        {
            Bar5h.Value = first.Utilization;
            Text5h.Text = $"{first.Utilization:0}%  {TimeFormatter.FormatResetTime(first.ResetsAt)}";
            SetBarColor(GetBarIndicator(Bar5h), first.Utilization);
        }

        if (second != null)
        {
            Bar7d.Value = second.Utilization;
            Text7d.Text = $"{second.Utilization:0}%  {TimeFormatter.FormatResetTime(second.ResetsAt)}";
            SetBarColor(GetBarIndicator(Bar7d), second.Utilization);
        }
    }

    public void ShowLoadingState()
    {
        Bar5h.Value = 0; Bar7d.Value = 0;
        Text5h.Text = SpinnerFrames[_spinnerFrame];
        Text7d.Text = SpinnerFrames[_spinnerFrame];
    }

    public void AdvanceSpinner()
    {
        _spinnerFrame = (_spinnerFrame + 1) % SpinnerFrames.Length;
        Text5h.Text = SpinnerFrames[_spinnerFrame];
        Text7d.Text = SpinnerFrames[_spinnerFrame];
    }

    public void ShowErrorState()
    {
        Bar5h.Value = 100; Bar7d.Value = 100;
        var maroon = new SolidColorBrush(Colors.Maroon);
        var ind5h = GetBarIndicator(Bar5h);
        var ind7d = GetBarIndicator(Bar7d);
        if (ind5h != null) ind5h.Background = maroon;
        if (ind7d != null) ind7d.Background = maroon;
        Text5h.Foreground = Brushes.White; Text7d.Foreground = Brushes.White;
        Text5h.Text = "Error"; Text7d.Text = "Error";
    }

    public void RefreshText(UsageData? lastUsage)
    {
        if (lastUsage == null) return;
        var first = lastUsage.Limits.ElementAtOrDefault(0);
        var second = lastUsage.Limits.ElementAtOrDefault(1);
        if (first != null)
            Text5h.Text = $"{first.Utilization:0}%  {TimeFormatter.FormatResetTime(first.ResetsAt)}";
        if (second != null)
            Text7d.Text = $"{second.Utilization:0}%  {TimeFormatter.FormatResetTime(second.ResetsAt)}";
    }

    private static void SetBarColor(Border? indicator, double utilization)
    {
        if (indicator == null) return;
        var color = utilization < 75 ? "#4CAF50" : utilization < 90 ? "#FF9800" : "#F44336";
        indicator.Background = new SolidColorBrush((Color)ColorConverter.ConvertFromString(color));
    }

    private static Border? GetBarIndicator(ProgressBar bar)
    {
        bar.ApplyTemplate();
        return bar.Template.FindName("PART_Indicator", bar) as Border;
    }
}
```

**MainWindow.xaml.cs** — změny:

1. Přidat field pro seznam panelů a klientů:
```csharp
private readonly List<(ClaudeApiClient Client, AccountPanel Panel)> _accounts = [];
```

2. Změnit konstruktor — přijímat `List<ClaudeApiClient>` místo `ClaudeApiClient`:
```csharp
internal MainWindow(List<ClaudeApiClient> clients, IntPtr taskbarHwnd, bool isPrimary)
{
    _taskbarHwnd = taskbarHwnd;
    _isPrimary = isPrimary;
    InitializeComponent();
    MouseEnter += (_, _) => ShowPopup();
    MouseLeave += (_, _) => HidePopup();

    // Vytvořit AccountPanel pro každý client
    foreach (var client in clients)
    {
        var panel = new AccountPanel(client.AccountService);
        AccountsPanel.Children.Add(panel);
        _accounts.Add((client, panel));
    }

    // Nastavit šířku okna dynamicky (COL_WIDTH × počet účtů)
    const double ColWidth = 170.0;
    Width = ColWidth * clients.Count;

    Loaded += async (_, _) => { /* stejný Loaded handler jako dříve, ale bez Bar5h/Bar7d přímých referencí */ };
}
```

3. Odebrat fields `_apiClient` (single client) — nahradit `_accounts` listem.

4. Upravit `Loaded` handler — pro každý `(client, panel)` v `_accounts`:
   - `panel.ShowLoadingState()`
   - `await client.GetUsageAsync()` → `panel.UpdateBars(usage)` nebo `panel.ShowErrorState()`

5. Upravit `StartRefreshTimer()` — iterovat přes `_accounts`, pro každý volat API a aktualizovat panel.

6. Upravit `StartTextTimer()` — iterovat přes `_accounts`, volat `panel.RefreshText(lastUsage)`. Uložit `_lastUsage` per-account v tuples nebo jako `Dictionary<AccountPanel, UsageData?>`.

7. Upravit `ShowPopup()` a `_lastUsage` — popup zobrazit pro první účet nebo pro hovered panel. Nejjednodušší: popup zůstane pro první účet (`_accounts[0]`), protože `PopupWindow` existuje beze změny.

8. `PositionWindow()` — kód zůstává beze změny. `Width` je nyní dynamicky nastaveno v konstruktoru, takže `int widgetWidthPx = (int)(Width * scale)` automaticky vrátí správnou hodnotu.

9. Odebrat `UpdateBars`, `ShowErrorState`, `ShowLoadingState`, `RefreshText`, `GetBarIndicator`, `SetBarColor`, `StopSpinner`, `ShowLoadingState` z MainWindow — tyto metody jsou teď na AccountPanel.

10. `_spinnerTimer` přesunout — spinner tick bude iterovat přes `_accounts` a volat `panel.AdvanceSpinner()`.
