<!-- Win32 API potřebné pro pozicování -->

SHAppBarMessage — zjistí pozici a rozměry taskbaru:
```csharp
[DllImport("shell32.dll")]
static extern uint SHAppBarMessage(uint dwMessage, ref APPBARDATA pData);

const uint ABM_GETTASKBARPOS = 0x00000005;

[StructLayout(LayoutKind.Sequential)]
struct APPBARDATA
{
    public uint cbSize;
    public IntPtr hWnd;
    public uint uCallbackMessage;
    public uint uEdge;
    public RECT rc;
    public int lParam;
}

// uEdge hodnoty: ABE_LEFT=0, ABE_TOP=1, ABE_RIGHT=2, ABE_BOTTOM=3

[StructLayout(LayoutKind.Sequential)]
struct RECT { public int Left, Top, Right, Bottom; }
```

FindWindow pro získání HWND system tray oblasti:
```csharp
[DllImport("user32.dll")]
static extern IntPtr FindWindow(string lpClassName, string lpWindowName);

// Shell_TrayWnd = hlavní taskbar okno
// TrayNotifyWnd = system tray area (child of Shell_TrayWnd)
```

GetWindowRect pro rozměry:
```csharp
[DllImport("user32.dll")]
static extern bool GetWindowRect(IntPtr hWnd, out RECT lpRect);
```

SystemParameters.WorkArea — oblast obrazovky bez taskbaru (WPF, DIP jednotky)
SystemParameters.PrimaryScreenHeight — celková výška obrazovky (WPF DIP)

DŮLEŽITÉ: Win32 pracuje s fyzickými pixely, WPF s DIP. Převod:
- DPI faktor: `PresentationSource.FromVisual(this).CompositionTarget.TransformFromDevice.M11`
- fyzické pixely → WPF DIP: `pixels * dpiScaleFactor`

Rozšířit MainWindow.xaml.cs o Win32 P/Invoke deklarace a metody pro pozicování.

1. Přidat using: `using System.Runtime.InteropServices;`

2. Přidat P/Invoke deklarace jako private static metody ve třídě MainWindow:
   - `SHAppBarMessage` s `APPBARDATA` structem a `RECT` structem
   - `FindWindow`
   - `GetWindowRect`
   - Konstanty: `ABM_GETTASKBARPOS = 0x5u`, `ABE_BOTTOM = 3u`

3. Přidat metodu `GetTaskbarInfo()` → vrátí `(bool isBottom, RECT taskbarRect)`:
   - Zavolá `SHAppBarMessage(ABM_GETTASKBARPOS, ref data)`
   - Vrátí `isBottom = data.uEdge == ABE_BOTTOM`, `taskbarRect = data.rc`

4. Přidat metodu `GetTrayWidth()` → vrátí `int` (šířka tray oblasti v fyzických pixelech):
   - `FindWindow("Shell_TrayWnd", null)` → hwndTaskbar
   - `FindWindowEx(hwndTaskbar, IntPtr.Zero, "TrayNotifyWnd", null)` → hwndTray
   - `GetWindowRect(hwndTray, out RECT trayRect)` → šířka = `trayRect.Right - trayRect.Left`
   - Fallback pokud HWND nenalezen: vrátit 200 (rozumný default pro Win11 tray)

   Přidat také `FindWindowEx`:
   ```csharp
   [DllImport("user32.dll")]
   static extern IntPtr FindWindowEx(IntPtr hwndParent, IntPtr hwndChildAfter,
       string lpszClass, string? lpszWindow);
   ```

5. Přidat metodu `PositionWindow()`:
   - Zavolat `GetTaskbarInfo()` — pokud `!isBottom`: zobrazit `MessageBox.Show("smůla bejku", "Claude Usage Widget")` a zavolat `Application.Current.Shutdown(); return;`
   - Získat DPI scale factor: `var source = PresentationSource.FromVisual(this); double scale = source?.CompositionTarget?.TransformFromDevice.M11 ?? 1.0;`
   - Šířka tray v DIP: `trayWidthDip = GetTrayWidth() * scale`
   - Šířka okna: `this.Width` (nastavená v XAML/konstruktoru, např. 120)
   - Pozice X = `SystemParameters.PrimaryScreenWidth - trayWidthDip - this.Width`
   - Pozice Y = `SystemParameters.WorkArea.Bottom` (spodní hrana WorkArea = top taskbaru)
   - Nastavit: `this.Left = posX; this.Top = posY;`

6. V konstruktoru MainWindow zavolat `PositionWindow()` — PO `InitializeComponent()` a PO `SetTaskbarHeight()`.

7. Přidat timer pro sledování změn tray šířky — `DispatcherTimer` s intervalem 2 sekundy:
   - Uložit `_lastTrayWidth` při posledním `PositionWindow()` volání
   - V timer ticku: `if (GetTrayWidth() != _lastTrayWidth) PositionWindow();`
   - Timer spustit v konstruktoru po `PositionWindow()`

Pozor: `PositionWindow()` volat jen po `Loaded` eventu nebo z konstruktoru po `Show()` — okno musí být v DOM pro `PresentationSource.FromVisual`. Nejbezpečnější: přesunout volání `PositionWindow()` do `Loaded` event handleru:
```csharp
Loaded += (_, _) => {
    SetTaskbarHeight();
    PositionWindow();
    StartTrayWatchTimer();
};
```
