# Plan 05-01: Embedded ikony služeb

## Objective

Přidat PNG ikony pro Claude a Codex jako embedded resources do projektu. Ikony budou použity v Plan 02 jako vizuální identifikátory služby vlevo od progress barů.

## Context

XAML v MainWindow.xaml momentálně nezobrazuje žádnou ikonu. Phase 5 vyžaduje ikonu identifikující službu (Claude logo / Codex logo) vedle každé sady barů (UI-07). Ikony musí být embedded v exe — ne externí soubory.

Ikony budou simple mini PNG 20×20px. Protože nejsou k dispozici oficiální loga jako soubory, vytvoříme minimalistické PNG ikonky přímo kódem nebo použijeme placeholder PNG (jednobarevný čtverec s písmenem). Důležité je správně nastavit Build Action a zpřístupnit je z XAML.

## Tasks

<task id="05-01-01">
**Vytvořit PNG ikony a přidat do projektu jako Resource**

1. Vytvořit adresář `ClaudeUsageWidget/Assets/`

2. Vytvořit dvě PNG ikony (20×20px) jako minimalistické placeholdery pomocí C# scriptu nebo ručně:
   - `claude-logo.png` — oranžový čtverec (#CC7000) s bílým "C"
   - `codex-logo.png` — modrý čtverec (#0066CC) s bílým "X"

   Ikony lze vytvořit jednorázovým C# scriptem (lze spustit jako `dotnet script` nebo přidat jako temp Console project) nebo stáhnout existující veřejně dostupná loga. Pokud není k dispozici žádný generátor, vytvořit minimální validní PNG ručně (binary PNG 20×20 jednobarevný — viz formát níže).

   **Alternativa — vygenerovat PNG přímo PowerShellem** (preferováno, funguje na Windows):
   ```powershell
   Add-Type -AssemblyName System.Drawing

   function New-IconPng($path, $bgColor, $letter) {
       $bmp = New-Object System.Drawing.Bitmap(20, 20)
       $g = [System.Drawing.Graphics]::FromImage($bmp)
       $g.Clear($bgColor)
       $font = New-Object System.Drawing.Font("Arial", 11, [System.Drawing.FontStyle]::Bold)
       $brush = [System.Drawing.Brushes]::White
       $sf = New-Object System.Drawing.StringFormat
       $sf.Alignment = [System.Drawing.StringAlignment]::Center
       $sf.LineAlignment = [System.Drawing.StringAlignment]::Center
       $g.DrawString($letter, $font, $brush, [System.Drawing.RectangleF]::new(0,0,20,20), $sf)
       $bmp.Save($path, [System.Drawing.Imaging.ImageFormat]::Png)
       $g.Dispose(); $bmp.Dispose()
   }

   $assetsDir = "ClaudeUsageWidget\Assets"
   New-Item -ItemType Directory -Force -Path $assetsDir | Out-Null
   New-IconPng "$assetsDir\claude-logo.png" ([System.Drawing.Color]::FromArgb(204,112,0)) "C"
   New-IconPng "$assetsDir\codex-logo.png"  ([System.Drawing.Color]::FromArgb(0,102,204))   "X"
   Write-Host "Icons created."
   ```
   Spustit jako: `powershell.exe -ExecutionPolicy Bypass -File create-icons.ps1` z kořene repozitáře.

3. V `ClaudeUsageWidget.csproj` přidat do `<ItemGroup>`:
   ```xml
   <Resource Include="Assets\claude-logo.png" />
   <Resource Include="Assets\codex-logo.png" />
   ```
   Build Action `Resource` (ne `EmbeddedResource`) zpřístupní soubory přes Pack URI: `pack://application:,,,/Assets/claude-logo.png`

4. Ověřit `dotnet build ClaudeUsageWidget/ClaudeUsageWidget.csproj` projde bez chyby.
</task>
